# Guitar Hero HUD

A simple 2D Pygame implementation of a Guitar Hero-style Heads-Up Display (HUD) that processes WAV audio files for note timing simulation.

## Features
- 4 lanes with colored notes
- Keyboard controls (A, S, D, F)
- Basic scoring
- Placeholder WAV beat detection

## Requirements
- Python 3
- Pygame: `pip install pygame`

## Running
```bash
python guitar_hero_hud.py
```

Note: Add your own .wav file for better integration. Currently simulates beats.
